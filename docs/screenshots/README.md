# Capturas recomendadas para README

Esta carpeta esta reservada para capturas reales de la aplicacion. Cuando se
quieran documentar visualmente las secciones del proyecto, guardar aqui las
imagenes con estos nombres:

- `01-login.png`: inicio de sesion y acceso con Google.
- `02-home-swipe.png`: pantalla principal con tarjetas.
- `03-mapa-publicaciones.png`: mapa de publicaciones.
- `04-chatbot.png`: asistente ConVive.
- `05-mis-publicaciones.png`: publicaciones del usuario.
- `06-panel-admin.png`: panel administrativo.
- `07-notificaciones.png`: notificaciones y estados.

Luego se pueden enlazar desde el README principal con Markdown:

```md
![Login de ConVive](docs/screenshots/01-login.png)
![Chatbot ConVive](docs/screenshots/04-chatbot.png)
```
