# Indice de Documentacion

Esta es la documentacion vigente del proyecto ConVive.

## Documentos Principales

| Archivo | Proposito |
| --- | --- |
| `README.md` | Memoria tecnica general de la app, arquitectura, modulos y estado de implementacion. |
| `QUICK_START.md` | Guia rapida para ejecutar Flutter y backend en local. |
| `backend/README.md` | Detalle del backend FastAPI, endpoints y variables de entorno. |
| `DEPLOYMENT_PRODUCCION.md` | Guia para llevar backend y frontend a produccion. |
| `docs/Librerias_seleccionadas.md` | Librerias usadas en el proyecto. |
| `docs/Herramientas_seleccionadas.md` | Herramientas usadas en el desarrollo. |

## Lectura Recomendada Para Tesis

1. `README.md`: vision completa del sistema.
2. `docs/Librerias_seleccionadas.md`: justificacion de librerias.
3. `docs/Herramientas_seleccionadas.md`: herramientas de desarrollo.
4. `backend/README.md`: descripcion tecnica del servicio IA.
5. `DEPLOYMENT_PRODUCCION.md`: despliegue y consideraciones de produccion.

## Documentos Eliminados

Se eliminaron documentos antiguos de arreglos puntuales, sesiones anteriores y
guias duplicadas. Su contenido relevante fue consolidado en los documentos
vigentes. Esto evita contradicciones y facilita presentar el proyecto como un
trabajo final ordenado.
